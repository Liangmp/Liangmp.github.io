My Second Semester in HKU

Posted on 2019-06-04 | Post modified: 2019-09-30 | In 随笔 |


趁短暂的summer break赶紧写个回顾吧。

有了第一个学期的经验，这学期对于assignment和group project的应付相对自如。这学期选的课都比较硬核，除了visualization and visual analysis比较水之外。本来选了五门课的，但是user interface design上完第一次课就机智地drop了，这是一门文科性质的课，一堆阅读材料，要写很多文字。

Deep Learning

这门课由两位老师一起教，前半部分的 introduction 由 Dr. Bethany M.Y. Chan 来讲，后半部分讲 Natural Language Processing (with RNNs), Image Processing (CNNs) and other applications, others such as: Unsupervised Feature Learning, Deep Generative Models, Deep Reinforcement Learning。这部分由一位法国过来的客坐讲师 Dr. Mauro Sozio 来教。Bethany 讲的东西有点浅，可能是因为我们上学期的artificial intellengence and machine learning 讲过了。Bethany 讲话声音较小，需要坐在前排才能听清楚，而且互动不多，所以一个不留神没跟上她讲的就很容易走神。而且 Bethany 讲话音调较平，她讲的内容又学过，而且我选的还是早上的课，所以听她的课没有咖啡肯定会犯困。相比之下，Mauro Sozio的课就好多了，他讲的很有逻辑，喜欢上课提问，可惜学生不是很积极，有时候提问没人回答，能看出他挺失落的。Bethany讲完她那部分之后有一个midterm test，占15%，全是计算题，我的midterm考得不好，有好几题都算错了。期末占35%，主观题比较多。这门课比较神奇，两个assignment分别占25%， 第一个assignment是关于rnn的，非常简单，第二个assignment是cnn的实现和优化，不难，但是做起来很花时间，还很耗电脑资源，反正我的小本本mbp跑起来的时候一直嗡嗡响个不停，还特别烫手。

Assignment 1 ………. 25%
Assignment 2…………25%
Midterm quiz………….15%
Final exam……………35%

Cluster and Cloud Computing

这门课的内容非常多，讲了Hadoop MapReduce, Spark, virtualization technique (which is one of the hardest part in this course), cluster network, and data center networking。有一个个人assignment和一个小组的group project，个人assignment是利用 AWS EC2 来部署一个bookstore，而group project 是四个人一个小组利用学院的机子diy一个小小的cluster，在上面部署一个跑在hadoop或者spark上的应用。有三个workshop，第一个是讲怎么用AWS EC2和AWS其他云产品的，第二个讲如何搭建一个cluster，如何用xen这个linux上的开源虚拟工具来虚拟出VM，用ganglia管理一个集群。第三个workshop讲hadoop和spark的安装和使用。TA们给workshop做的tutorial PPT都非常详细，但是跟着做的时候还是会遇到各种各样奇奇怪怪得问题，所以完成这几个workshop是非常耗时间和精力的。最费劲的还是group project，单是Prof.C.L.Wang给出的project guide就有31页这么长，里面写满了各种各样非常详细的要求。很意外的是，我们的group project得到了Wang的认同，拿到了A+，非常感谢我的队友。

Visualization and Visual

主要讲数据可视化以及可视化的工具，非常简单，几乎啥都没学到，要不是没其他课的可选我肯定会退掉这门。由于太水，我敲掉了好多次课和lab。又由于过于不重视这门课，我们group这门课的final project居然是集体通宵一个晚上做完的，还被CS lab罚了quota(we violated a rule in the lab)。Anyway，这是我们group通宵一个晚上的成果：

Advanced Database Management

很多同学选这门课是为了蹭个学分，并且悠闲地写sql语句做增删改查，结果谁也没想到这门课真如课程名称一样advanced。相信大家上完第一节课都有跟我一样的感受，怎么一个introduction能讲这么多东西？事实上这个intro就是这门课所涵盖的内容，包括：

  1. Data storage
    • Disk and disk management
    • files, pages, records
    • file management (heap file, sorted file and their cost of operations)
  2. Tree indexes
    • ISAM(Indexed Sequential Access Method)
    • B+ tree
    • Clustered and unclustered indexes
  3. Buffer management
    • Handling dirty pages
    • Page replacement policy
      • LRU (Least-recently-used, may cause “sequential flooding” in some scenario)
      • MRU (Most-recently-used, good for repeated scan)
      • Clock policy (a simplify version of LRU)
  4. Sorting and hashing (I have to say that is part is very hard)
    • Out-of-core algorithms (sort 100GB of data with 1GB of RAM)
    • Sorting: conquer and merge
    • Hashing: divide and conquer
  5. Relational algebra
  6. Interators joins
  7. Query optimization
  8. Concurrency control and transactions
  9. Crash recovery
  10. Parallel DBMS
  11. NoSQL
  12. BDMS (Big Data Management System)
  13. Column oriented DBMS (Not discussed this year due to time limitation)
