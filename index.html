Linux Note

Posted on 2020-03-01 | Post modified: 2020-03-01

Linux is an operating system or a kernel. It is distributed under an open source license. Its functionality list is quite like UNIX. Linux is a clone of UNIX. The commands used on both the operating systems are usually the same. So learning one is same as learning another.

Here is the story, if you are interested in stories. Back in the 1990s (maybe a few years earlier), a computer science master student Linus Torvalds thought that the UNXI OS needed to be improved. After his suggestion was rejected by the designers of UNXI, he thought of launching an OS which will be receptive to changes, modifications suggested by its users. Linus Torvalds named the new kernel Linux, and the first release was in March 1991. Now, Linux is one of the most popular and widely used Kernel, and it is the backbone of popular operating systems like Debian, Knoppix, Ubuntu, and Fedora. The main advantage of Linux was that programmers were able to use the Linux Kernel to design their own custom operating systems.

Do you know that Linus Torvalds is also the original auther of the famous distributed version control system Git? When being ask why the name “git”, “Git in British slang means ‘a rotten person’,”he said, “I’m an egotistical bastard, so I name all my projects after myself. First Linux, now git.”
