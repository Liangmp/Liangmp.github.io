Chrome+SwitchyOmega+Shadowsocks科学上网

Posted on 2017-11-26 | Post modified: 2019-01-22 | In 技术 |

世界这么大，我想去看看


个人认为，在大陆不会科学上网，真的会错过太多太多的耗资源。一位朋友使用了vpn后感叹了一句，感觉自己出国了（大笑.jpg）。是的，我想告诉你google搜索比百度搜索好用一百倍，个人认为百度之所以能在国内存活这么多年，完全是因为google退出了中国市场。在没有vpn的情况下，宁愿用bing也不用百度。下面说一下windows如何用chrome+SwitchyOmega+ShadowSocks进行科学上网。

  1. 买一个vpn（或者找一个免费的vpn也行），付费vpn推荐我目前使用的vpn，叫一枝红杏，免费vpn可参考免费ShadowSocks账号
  2. 下载一个chrome插件，叫SwithcyOmega。本来是可以直接通过google浏览器安装的，但是由于GFW限制，目前在国内只能通过百度网盘下载SwithcyOmega安装包手动安装了。下载链接：http://pan.baidu.com/s/1pKXAGEf 密码：8n4d
  3. 安装SwitchyOmega，安装方式简单粗暴，打开Chrome，把SwitchyOmega直接拖到浏览器扩展程序页面里面就OK了。
  4. 设置SwitchyOmega，设置情景模式实现自动切换。具体操作请参考GFWList。
  5. 下载shadowsocks。推荐使用最新版的客户端，如果出现打开后直接显示程序未响应或停止工作等错误，请尝试历史版本，如3.3.2版本或者2.5.8的版本，
    如果客户端程序运行有报错，3.3.5及其以上版本需要.net framework 4.6.2运行库，如果您未安装可以在这里下载：链接: http://pan.baidu.com/s/1i54qRDr 密码: xemi，
    3.3.5以下版本请安装.net framework 4.0后再尝试运行！下载链接: 链接: http://pan.baidu.com/s/1bYbkpK 密码: rt55。打开shadowsocks，填入购买的IP地址，端口号，密码，加密方式。

如果你是Mac系统，那么恭喜你，在Mac下使用vpn极其方便，因为GUI已经集成了系统 Pac，所以无需设置任何科学上网的姿势，直接运行ShadowsocksX ，然后输入的账号信息，保存运行后，即可在所有浏览器里直接科学上网。MAC OS X 客户端：链接: http://pan.baidu.com/s/1cGB61o 密码: 9dst
