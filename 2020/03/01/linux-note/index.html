Linux Note

Posted on 2020-03-01 | Post modified: 2020-04-16 |

Linux is an operating system or a kernel. It is distributed under an open source license. Its functionality list is quite like UNIX. Linux is a clone of UNIX. The commands used on both the operating systems are usually the same. So learning one is same as learning another.

Here is the story, if you are interested in stories. Back in the 1990s (maybe a few years earlier), a computer science master student Linus Torvalds thought that the UNXI OS needed to be improved. After his suggestion was rejected by the designers of UNXI, he thought of launching an OS which will be receptive to changes, modifications suggested by its users. Linus Torvalds named the new kernel Linux, and the first release was in March 1991. Now, Linux is one of the most popular and widely used Kernel, and it is the backbone of popular operating systems like Debian, Knoppix, Ubuntu, and Fedora. The main advantage of Linux was that programmers were able to use the Linux Kernel to design their own custom operating systems.

Do you know that Linus Torvalds is also the original auther of the famous distributed version control system Git? When being ask why the name “git”, “Git in British slang means ‘a rotten person’,”he said, “I’m an egotistical bastard, so I name all my projects after myself. First Linux, now git.”

differences among yum, rpm, pip, apt, apt-get

YUM: Yellowdog Updater Modified, an open source command-line, and a graphical based package management tool for RPM (RedHat Package Manager) based Linux systems. It allows users and system administrator to easily install, update, remove or search software packages on a systems.
ref: 20 Linux YUM (Yellowdog Updater, Modified) Commands for Package Management
YUM COMMAND CHEAT SHEET

RPM: RedHat Package manager, an default open source and most popular package management utility for Red Hat based systems like (RHEL, CentOS and Fedora).

pip: pip is the package installer for python.

apt: a command-line utility for installing, updating, removing and otherwise managing deb packages on Ubuntu, Debian, and related Linux distributions. It combines the most frequently used commands from the apt-get and apt-cache tools with different default values of some options.

The powerful editor: vim

Move left, right, up, down:
j // down
k // up
h // left
l // down

Move to start/end of line:
$ // end of line
0 // (zero) start of line

Insert mode:
i // insert
a // append
o // open (append) blank line below current line (no need to press return)
O // open blank line above current line
esc // exit insert mode

x // delete charactor
d // delte a word
dd // delete current line
yy // yank(copy) a line
p // put (paste) the clipboard after cursor
P // put (paste) before c

See Why yank is the meaning of copy in vim - stackoverflow

:w // write (save) the file, but don’t exit
:wq // write (save) and quit
:q // quit (fails if anything has changed)
:q! // quit and throw away changes

/pattern // search for pattern
?pattern // search backward for pattern
n // repeat search in same direction
N // repeat search in opposite direction
:%s/old/new/g // replace all old with new throughout file
:%s/old/new/gc // replace all old with new throughout file with confirmations
